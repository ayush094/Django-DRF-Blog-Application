┌─────────────────────────────────────────────────────────────────────────┐
│  TROUBLESHOOTING                                                       │
│                                                                         │
│  ❌ 401 Unauthorized                                                   │
│      → Ensure you're logged in. JWT token in localStorage?            │
│                                                                         │
│  ❌ Audio not generating                                              │
│      → Check Celery worker is running                                │
│      → Verify ANTHROPIC_API_KEY and GOOGLE_APPLICATION_CREDENTIALS  │
│      → Check Celery logs for errors                                   │
│                                                                         │
│  ❌ Static files not loading                                          │
│      → Run: python manage.py collectstatic                            │
│      → Check STATIC_URL in settings.py                                │
│                                                                         │
│  ❌ 404 on /audio-hub/                                                │
│      → Did you add AudioHubView to blog_app/urls.py?                 │
│      → Did you include blog_app.urls in myproject/urls.py?           │
│      → Migrations applied?                                           │
│                                                                         │
│  ❌ Player shows but audio won't play                                │
│      → Check if audio_url is valid in browser console                │
│      → CORS issues if using different domain for media               │
│      → Verify MEDIA_URL and MEDIA_ROOT configured                    │
└─────────────────────────────────────────────────────────────────────────┘

│  CUSTOMIZATION                                                        │
│                                                                         │
│  Change Color Scheme:                                                 │
│    Edit blog_app/static/audio/css/audio-styles.css                   │
│    Update :root { --md-primary: #YOUR_COLOR; }                       │
│                                                                         │
│  Add Language:                                                        │
│    Edit blog_app/audio/models.py → LANGUAGE_CHOICES                  │
│    Add <option> to audio_hub.html                                    │
│                                                                         │
│  Modify Player Layout:                                                │
│    Edit audio-app.js → renderPlayer() method                         │
│                                                                         │
│  Add New Features:                                                    │
│    Extend AudioBlogApp class in audio-app.js                         │
│    Add endpoints to audio/urls.py                                     │
